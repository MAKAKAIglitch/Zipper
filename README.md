**Zipper** is a lightweight, robust batch script wrapper for the Windows `compact.exe` utility. It allows you to transparently compress game folders and applications to save disk space using the LZX algorithm.

**Features:**
*   **GUI Folder Selection:** No need to copy-paste paths manually.
